# 👜 Zee by Zunaisha

A full-stack MERN eCommerce application for premium bags with secure authentication, admin product management, shopping cart, checkout, and Cloudinary image uploads.

## video walk through

🌐 [https://youtu.be/oPiIx26dFYM]

## ✨ Features

* User Registration & Login
* Admin Dashboard
* Product Management (CRUD)
* Shopping Cart
* Checkout & Order Management
* Cloudinary Image Upload
* JWT Authentication
* Responsive Design

## 🛠 Tech Stack

* React + Vite
* Tailwind CSS
* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* Cloudinary
* JWT

## ⚙️ Installation

```bash
git clone <YOUR_REPO_URL>
cd project
npm install
```

Create the required `.env` files (see `.env.example`) and run:

```bash
# Backend
cd backend
npm run dev

# Frontend
cd frontend
npm run dev
```
